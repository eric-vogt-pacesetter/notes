# Language Details

## Null and Undefined
```
let a; // a === undefined
a = null; // can set a to null
```

## Operators
```
// Spread Operator
let newArray = [...array, newNode]

// Strict equality
log('1' === 1) // false
log('1' == 1) // true
log('1' !== 1) // true
log('1' != 1) // false
```
